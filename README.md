# Documentação do Curso de Estrutura de Dados

## Visão Geral do Projeto
Este projeto tem como objetivo fornecer um guia abrangente sobre várias estruturas de dados e suas implementações. O curso abrange conhecimento teórico, aplicações práticas e diretrizes de projeto para capacitar os alunos com habilidades essenciais em manipulação de dados e otimização de algoritmos.

## Tópicos Abordados
- [Introdução às Estruturas de Dados](https://github.com/olooolll/dataStruct/tree/main/TAD)
- Arrays e Strings
- Listas Ligadas (Encadeadas)
- Pilhas
- Filas
- Árvores
- Grafos
- Tabelas Hash
- Algoritmos de Ordenação
- Algoritmos de Busca
- Análise de Complexidade

## Implementações de TAD (Tipos Abstratos de Dados)
1. **Arrays**: Implementação de arrays dinâmicos e estáticos com operações como inserção, remoção e acesso.
2. **Listas Ligadas**: Implementações de listas encadeadas simples e duplas com vários métodos de travessia e manipulação.
3. **Pilhas**: Implementação de pilha com operações de empilhar (*push*), desempilhar (*pop*) e espiar (*peek*).
4. **Filas**: Implementação de fila com operações de enfileirar (*enqueue*), desenfileirar (*dequeue*) e frente (*front*).
5. **Árvores**: Implementações de árvore binária e árvore binária de busca com algoritmos de travessia.
6. **Grafos**: Representação de grafos usando matriz de adjacência e lista de adjacência, com métodos de travessia como DFS (Busca em Profundidade) e BFS (Busca em Largura).
7. **Tabelas Hash**: Implementação de tabelas hash com técnicas de resolução de colisão.

## Estrutura
O curso está estruturado em múltiplos módulos, cada um focado em uma estrutura de dados específica, fornecendo embasamento teórico seguido de sessões de codificação e trabalho de projeto.
